# Nixpkgsとは
*NixOS/nixpkgs*というGithubリポジトリには各種パッケージのビルド情報からNix言語で利用できるライブラリなど様々な情報が集約されている。

| ブランチ             | 更新タイミング  | 利用対象    |
| ---------------- | -------- | ------- |
| master           | 適宜(最新)   | -       |
| nixpkgs-unstable | CI/CD完了後 | NixOS以外 |
| nixos-unstable   | CI/CD完了後 | NixOS   |
| nixpkgs-25.11    | 半年ごと     | NixOS以外 |
| nixos-25.11      | 半年ごと     | NixOS   |
## パッケージがあるか調べる
[NixOS Search](https://search.nixos.org/packages)
