# シェルの設定
利用しているシェルにて Home Manager が作成した環境変数を利用可能にするため、以下を `~/.profile`（Bash）や `~/.zprofile`（Zsh）等に追記してください
```zsh
. "$HOME/.nix-profile/etc/profile.d/hm-session-vars.sh"
```
**補足**
```zsh
echo $HOME
/Users/endouikki
```
## 利用するNixpkgsのブランチ変更
flake.nix内の*inputs*ではNixpkgsのnixos-unstableブランチが指定されている。
なので、変更する。
```flake.nix
inputs = { 
		# Specify the source of Home Manager and Nixpkgs. 
-		nixpkgs.url = "github:nixos/nixpkgs/nixos-unstable";
+		nixpkgs.url = "github:nixos/nixpkgs/nixpkgs-unstable"; 
		home-manager = { 
			url = "github:nix-community/home-manager"; 
			inputs.nixpkgs.follows = "nixpkgs"; 
		}; 
	}; 
```
```zsh
home-manager switch --flake .
```
### 設定ファイルの配置場所を変更
`flake.nix` には入力情報の定義と各ツール（Home Manager 含む）のオーケストレーションを担わせます。  
`home.nix` などの `*.nix` ファイルには具体的なユーザー環境に入れるパッケージのリストなどの情報を持たせます。
```tree
. ├── flake.lock
  ├── flake.nix
  └── home-manager
       └── home.nix
```
```flake.nix
homeConfigurations."endouikki" = 
		home-manager.lib.homeManagerConfiguration { 
			inherit pkgs; 
-			modules = [ ./home.nix ]; 
+			modules = [ ./home-manager/home.nix ]; 
		}; 
```
```zsh
home-manager switch --flake .
```
